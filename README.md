# AnoMessenger
It's a simple client/server messenger app with encryption feature

# Starting the server
On Linux:
```
$: pip3 install flask
$: cd server
$: python3 app.py
```

On Windows:
'''
pip install flask
cd server
python app.py
```

# Starting the client
On Linux:
```
$: pip3 instal flask
$: cd client
$: flask run
```

On windows:
```
pip install flask
cd client
flask run
```
